4. 执行java代码

4.1. 准备工具

在Sublime中安装Package Resource Viewer
（1）cmd+shift+p打开package control；
（2）输入Install Package；

（3）搜索Package Resource Viewer并安装。如果已经安装就搜不到，但可以通过list package 查看

说明已经安装

4.2. 开始配置

(4) 修改Javac.sublime-build
安装Package Resource Viewer就是为这一步服务。同上面的步骤，打开package control，输入PackageResourceViewer:Open Resource

回车后输入java

回车后输入Javac.sublime-build,就能够打开我们所需要修改的文件

最后修改文件为：

{
"shell_cmd": "javac -encoding utf-8 $file_name && java $file_base_name  && rm -rf $file_base_name.class",
"file_regex": "^ *\\[javac\\] (.+):([0-9]+):() (.*)$",
"selector": "source.java",
"encoding": "utf-8"
}

上面执行java代码的步骤为：编译 -> 运行 -> 删除.class文件
可以根据自己的需要随意组合执行步骤。

5. 执行Python代码

5.1. 创建配置

Tools -> Build System -> New Build System

{
"cmd": ["/usr/local/pyenv/shims/python", "${file}"],
"file_regex": "^(..[^:]*):([0-9]+):?([0-9]+)?:? (.*)$",
"working_dir": "${file_path}",
"selector": "source.py",
"encoding": "utf-8",
"env": {"PYTHONIOENCODING": "utf-8"}
}

保存为python3.sublime-build

注意：
“env”: {“PYTHONIOENCODING”: “utf-8”}要加上，否则会报错：
UnicodeEncodeError: ‘ascii’ codec can’t encode characters in position

5.2. 应用配置

Tools -> Build System -> 选中python3

5.3. 编辑配置

cmd+shift+p打开package control，输入Browse Packages，选中python3.sublime-build文件编辑
